# AI Insights Dashboard

A modern AI-powered analytics dashboard built with React, TypeScript, Tailwind CSS, and FastAPI.  
Designed for showcasing Artificial Intelligence, data analytics, visualization, and real-time monitoring skills in a professional portfolio.

This project demonstrates enterprise-level frontend architecture, AI integration, interactive analytics, and modern UI/UX design suitable for internships and real-world applications.

Installation
Clone Repository
git clone https://github.com/yourusername/AI-Insights-Dashboard.git
cd AI-Insights-Dashboard
Frontend Setup
cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:5173
Backend Setup
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

Backend runs on:

http://localhost:8000
Environment Variables

Create .env files in frontend and backend folders.

Frontend .env
VITE_API_URL=http://localhost:8000
VITE_FIREBASE_API_KEY=YOUR_KEY
Backend .env
OPENAI_API_KEY=YOUR_KEY
DATABASE_URL=YOUR_DATABASE_URL
